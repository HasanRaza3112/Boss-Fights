# Boss-Fights

Boss-Fights is a Unity action game project focused on boss-only combat. The current MVP concept is an anime-stylized, Sekiro-inspired but fully original samurai duel game starring Akuza.

## Current Vision

- Genre: third-person action boss rush
- Engine: Unity
- Platforms: PC and mobile
- Hero: Akuza, a samurai
- MVP boss: human rival
- Art direction: anime-stylized 3D
- Release goal: portfolio-quality MVP with a path toward commercial publishing
- Core fantasy: precise sword duels built around deflect timing, posture pressure, readable attacks, and cinematic finishers

## MVP Scope

The first release should prove the core combat loop before expanding into more bosses.

Must have:
- One playable hero: Akuza
- One fully playable human rival boss
- One polished duel arena
- Health and posture systems
- Deflect, guard, dodge, attack, heal, and execution actions
- Three difficulty modes
- PC keyboard/mouse, controller, and mobile touch controls
- Result screen with time, deaths, damage taken, and deflect stats

Not in MVP:
- Open world
- Multiple heroes
- Large enemy roster
- Multiplayer
- Loot-heavy RPG progression
- Any copied Sekiro assets, names, animations, UI, audio, or lore

## Legal Direction

Sekiro is a design reference only. This project must use original characters, weapons, arenas, animation, VFX, audio, UI, lore, and names. Extracted game assets or unclear fan asset packs are not acceptable for commercial release.

## Recommended Unity Direction

- Unity 6 LTS or current stable Unity LTS
- Universal Render Pipeline
- Unity Input System
- Cinemachine
- Mobile-first performance budgets
- RAG-assisted design documentation for agent consistency
