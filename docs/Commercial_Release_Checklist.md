# Commercial Release Checklist

## 1. Product Definition

- [ ] Finalize game title.
- [ ] Finalize one-sentence pitch.
- [ ] Finalize Akuza visual identity.
- [ ] Finalize first rival boss name and identity.
- [ ] Finalize MVP arena theme.
- [ ] Decide target stores for first release.
- [ ] Decide whether MVP is free demo, paid release, or portfolio build.
- [ ] Decide monetization model if commercial.

## 2. Legal and IP Safety

- [ ] Confirm all character names are original.
- [ ] Confirm all lore is original.
- [ ] Confirm all 3D models are original or commercially licensed.
- [ ] Confirm all animations are original, commissioned, or commercially licensed.
- [ ] Confirm all music and SFX are original or commercially licensed.
- [ ] Confirm all UI icons and fonts are licensed.
- [ ] Keep license files and receipts in a tracked location.
- [ ] Do not use extracted Sekiro assets.
- [ ] Do not use unclear fan model packs.
- [ ] Do not use copied boss names, attack names, UI, level layouts, or exact animations.

## 3. Unity Project Setup

- [ ] Select Unity version.
- [ ] Set up URP.
- [ ] Set up Unity Input System.
- [ ] Set up Cinemachine.
- [ ] Set up version control friendly project settings.
- [ ] Create build targets for Windows and Android.
- [ ] Add automated backup or repo workflow.
- [ ] Confirm `.gitignore` excludes generated Unity folders.
